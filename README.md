
# noOneAlone - Next.js Multilingual Starter (Expanded)

This project is a starter Next.js App Router site with TailwindCSS and multilingual structure.
Replace the placeholder WhatsApp number in `app/[locale]/(site)/contact/page.tsx` with your number.

## Quick start

1. Install:
```
npm install
```

2. Run dev:
```
npm run dev
```

3. Open:
http://localhost:3000/en/(site)

## Deploy to Vercel
- Push to GitHub and connect repo to Vercel.
- Add custom domain in Vercel dashboard.
